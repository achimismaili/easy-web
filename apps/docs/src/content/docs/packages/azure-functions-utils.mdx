## Installation

```bash
pnpm add @achimismaili/easy-web-azure-functions-utils
```

## Status

**In development.** This package is currently a stub. No public API is exported yet.

`@achimismaili/easy-web-azure-functions-utils` is planned to provide server-side helpers for [Azure Functions](https://learn.microsoft.com/en-us/azure/azure-functions/) used alongside easy-web Astro sites. The planned scope includes:

- Request/response utilities compatible with Azure Functions v4 (Node.js)
- MSAL-based server-side token validation middleware
- Helpers for calling Microsoft Graph from a Function context without a browser
- CORS configuration helpers for Functions fronting Astro static sites

## Roadmap

Utilities will be added here as Azure Functions use cases are validated in the `dev.ismaili.de` pilot. Follow the [easy-web changelog](https://github.com/achimismaili/easy-web/blob/main/CHANGELOG.md) for release announcements.
