## Hero Component

The `Hero` component renders a full-width introductory section.

### Usage

```astro
---
import { Hero } from '@achimismaili/easy-web-content-blocks';
---
<Hero
  title="Welcome to My Site"
  subtitle="Built on the easy-web platform"
  cta={{ label: 'Get Started', href: '/docs' }}
/>
```

## ThemeToggle Component

The `ThemeToggle` component switches between light and dark themes using CSS custom properties from `@achimismaili/easy-web-theme-core`.

### Usage

```astro
---
import { ThemeToggle } from '@achimismaili/easy-web-content-blocks';
---
<!-- Place in your site Header -->
<ThemeToggle />
```

### How It Works

The toggle switches the `data-theme` attribute on `<html>` between `light` and `dark`. The theme CSS from `@achimismaili/easy-web-theme-core` uses this attribute to apply the correct CSS custom property values.

```css
/* theme-core applies tokens like this: */
[data-theme="dark"] {
  --ew-color-bg: #1a1a2e;
  --ew-color-surface: #16213e;
  /* ... */
}
```

## Section Component

The `Section` component wraps content in a semantically correct `<section>` element with consistent spacing.

### Usage

```astro
---
import { Section } from '@achimismaili/easy-web-content-blocks';
---
<Section title="About Us" id="about">
  <p>Content goes here...</p>
</Section>
```

## Using Components Together

Here is a typical page structure using multiple easy-web components:

```astro
---
import {
  Hero,
  Section,
  CardGrid,
  Card,
  Header,
  Footer,
  ThemeToggle
} from '@achimismaili/easy-web-content-blocks';
import '@achimismaili/easy-web-theme-core/theme.css';
---
<html>
  <body>
    <Header>
      <ThemeToggle slot="end" />
    </Header>

    <Hero
      title="My Site"
      subtitle="Powered by easy-web"
      cta={{ label: 'Get Started', href: '/docs' }}
    />

    <Section title="Features">
      <CardGrid>
        <Card title="Fast" description="Optimised for performance" />
        <Card title="Accessible" description="WCAG 2.1 compliant" />
        <Card title="Themeable" description="Light and dark mode built-in" />
      </CardGrid>
    </Section>

    <Footer />
  </body>
</html>
```
