## Packages

The easy-web library consists of several focused packages:

| Package | Description |
| ------- | ----------- |
| `@achimismaili/easy-web-theme-core` | CSS design tokens, light/dark theme, no-flash script |
| `@achimismaili/easy-web-i18n` | i18n helpers for Astro sites |
| `@achimismaili/easy-web-content-blocks` | Hero, Section, CardGrid, Card, Header, Footer, ThemeToggle |
| `@achimismaili/easy-web-auth` | MSAL.js authentication, Microsoft Graph integration |
| `@achimismaili/easy-web-cms-adapters` | Decap CMS integration for content management |
| `@achimismaili/create-easy-web` | Scaffold CLI for creating new site instances |
